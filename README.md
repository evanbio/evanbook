# Tessera

Tessera is the visual companion to biopalette — a curated space for palette previews, toy examples, and figure-oriented presentation.

The R package lives in [biopalette](https://github.com/evanbio/biopalette); this repository focuses on presentation, examples, and visual documentation.

## What's inside

- Palette registry
- Preview pages
- Toy-data demonstrations
- Applied figure examples
